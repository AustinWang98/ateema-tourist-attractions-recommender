## Rebuilding the paper

```bash
cd docs/paper
latexmk -pdf main.tex
# or:
pdflatex main.tex && pdflatex main.tex
```

`main.tex` uses `\graphicspath{{./figures/}}`, so it compiles in-place without copying assets around.

To regenerate the three custom diagrams (architecture, score blend, MMR):

```bash
cd docs/paper/figures
python3 gen_diagrams.py
```

## Rebuilding the presentation

The deck is regenerated by hand-authoring slides via `python-pptx` and a copy of the program-supplied `Capstone I Initial Prosentation.pptx` template (kept outside the repo so its original imagery is not redistributed).

```bash
cd docs/presentation
pip install python-pptx pillow matplotlib

# point at the template file the program provided
export CHICAGODOES_TEMPLATE=/path/to/Capstone\ I\ Initial\ Prosentation.pptx
python3 build_presentation.py
```

If `CHICAGODOES_TEMPLATE` is unset, the script falls back to the already-built `ChicagoDoes_Final_Presentation.pptx` in this folder so the visual master, layouts, and theme are still inherited.

To re-render the formula PNGs used on the algorithm-detail slide, see the inline matplotlib mathtext snippet in the commit history of `assets/formulas/`.
